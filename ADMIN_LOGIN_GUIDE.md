# FastAdmin 后台管理使用指南

## 一、后台登录

### 登录地址
```
http://你的域名/admin
```

### 默认管理员
- **用户名**: admin
- **密码**: 安装时设置的密码

### 忘记密码重置方法

#### 方法 1：数据库重置
```sql
-- 将密码重置为 123456
UPDATE `fa_admin` 
SET `password` = 'e10adc3949ba59abbe56e057f20f883e' 
WHERE `username` = 'admin';
```

#### 方法 2：命令行重置
```bash
# 进入项目根目录
php think command -c Admin -a resetpwd -u admin -p 123456
```

#### 方法 3：使用官方密码重置工具
访问：`http://你的域名/admin/index/resetpwd`

## 三、配置助力管理菜单

如果登录后看不到"助力管理"菜单，需要手动添加：

### 步骤 1：进入菜单管理
后台 → 系统管理 → 菜单管理 → 添加

### 步骤 2：添加父级菜单
```
标题：助力管理
规则名称：boost
图标：fa fa-handshake-o
类型：目录
排序：100
```

### 步骤 3：添加子菜单

#### 助力任务
```
标题：助力任务
规则名称：boost/task
图标：fa fa-list
类型：菜单
父级菜单：助力管理
URL: boost/task/index
```

#### 任务分类
```
标题：任务分类
规则名称：boost/category
图标：fa fa-th-list
类型：菜单
父级菜单：助力管理
URL: boost/category/index
```

#### 助力订单
```
标题：助力订单
规则名称：boost/order
图标：fa fa-file-text
类型：菜单
父级菜单：助力管理
URL: boost/order/index
```

### 步骤 4：设置权限
在菜单管理中，为每个菜单勾选操作权限：
- ✅ index - 查看
- ✅ add - 添加
- ✅ edit - 编辑
- ✅ del - 删除
- ✅ audit - 审核（任务管理专用）
- ✅ detail - 详情

### 步骤 5：刷新权限
- 退出重新登录
- 或：后台 → 系统管理 → 刷新缓存

## 四、管理员管理

### 添加管理员
1. 后台 → 系统管理 → 管理员管理 → 添加
2. 填写用户名、密码、邮箱
3. 选择所属角色组
4. 设置可访问的权限

### 角色权限管理
1. 后台 → 系统管理 → 角色权限
2. 添加角色组（如：客服、运营、超级管理员）
3. 为角色分配菜单和操作权限

### 操作日志
后台 → 系统管理 → 操作日志
- 记录所有管理员的操作
- 可以查看、搜索、导出日志

## 七、安全建议

### 1. 修改默认管理员账号
- 修改 admin 用户名为其他名称
- 设置强密码（字母 + 数字 + 符号）

### 2. 限制登录 IP
在系统设置中配置允许登录的 IP 白名单

### 3. 开启二次验证
启用谷歌验证器或短信验证

### 4. 定期备份
- 定期备份数据库
- 备份代码文件

### 5. 权限最小化
- 给不同管理员分配最小必要权限
- 定期审查权限分配

## 八、常见问题

### Q1: 登录后看不到助力管理菜单？
A: 需要先在菜单管理中添加助力管理相关菜单，并分配权限。

### Q2: 如何给客服人员只分配查看权限？
A: 
1. 创建"客服"角色组
2. 只勾选"查看"权限，不勾选"添加"、"编辑"、"删除"等权限
3. 将客服人员分配到客服角色组

### Q3: 如何批量审核任务？
A: 在任务列表中勾选多个任务，然后点击"审核通过"或"审核拒绝"按钮。

### Q4: 后台登录地址可以修改吗？
A: 可以，修改 `application/route.php` 或服务器配置。

### Q5: 如何禁用某个管理员？
A: 在管理员管理中，将该管理员的状态设置为"禁用"。

## 九、后台功能扩展

### 添加新的管理模块
1. 创建控制器：`application/admin/controller/你的模块/YourController.php`
2. 创建视图：`application/admin/view/你的模块/`
3. 在菜单管理中添加菜单
4. 分配权限

### 使用 CRUD 生成器
FastAdmin 提供 CRUD 命令快速生成管理模块：
```bash
php think crud -t 表名
```

### 使用菜单命令
```bash
php think menu
```
